# EduClient Go 原生版

基于 Go 与 Fyne 的桌面端重写版本，主界面、成绩表、课程表和批量任务均为原生控件，不使用 Web 前端。

## 当前模块

- 登录会话：内置 WebView2 学校统一认证窗口，可选择教务系统直连或 WebVPN 线路；WebVPN 从 `https://webvpn.zstu.edu.cn/` 进入，将教务系统的新窗口链接收回内置窗口，并在跳转到 `jwglxt-443.webvpn.zstu.edu.cn` 教务首页后自动提取和校验 Cookie。
- 成绩查询：筛选学年/学期、成绩表、平均分和加权平均绩点、导出 CSV。
- 选课查询：获取课程分类、按关键词查询教学班、选择目标教学班。
- 批量抢课：逐行输入教学班名称；同一学年学期、同一大课代码的教学班会自动合并为一组，并按输入顺序作为候选优先级。前一候选满额后切换下一候选；任一候选成功即停止该组，其余大课组继续运行。支持单次请求超时、候选持续时间、线程间隔、循环模式和实时日志。

## 时间参数

- `单次超时(ms)`：每一次 HTTP 抢课请求允许等待的最长时间，默认 `15000`。
- `持续时间(s)`：一个候选教学班本轮并发轮询的时间窗口，默认 `30`；`-1` 表示不设置总时间上限，直到成功、确认满额后切换、会话停止或手动停止。
- `循环模式`：有限持续时间结束后，是否重新解析课程并开启下一轮；`持续时间=-1` 时无需依靠循环模式维持运行。

## 构建与启动

```powershell
cd D:\Project\EduClient\go-client
go build -trimpath -ldflags "-s -w -H=windowsgui" -o dist\EduClient-Go.exe .\cmd\educlient
.\dist\EduClient-Go.exe
```

也可以从项目根目录运行 `start-go.cmd`。
